这个仓库用来存一些自己平常鼓捣的小项目，这些项目大多与科研无关，许多是自己平时看书过程中一些突发奇想的点子，于是给它取了一个自认为很诗意的名字 **塞壬之歌** ，来比喻这些美丽而致命却无法令人抗拒的想法。我会不断完善这些项目，并及时整理已经完善的一些项目，一般会共享数据和代码，但有些数据因为有较高的私密性，不方便共享，但代码都会公开，由于好多都是研一写的，看起来比较笨。

目前主要包括四个小项目：

* 京剧数据的分析
* 唐代人物亲缘关系考
* 遗传所2017年入学学生的来源
* 遗传所自2010年起发表文章汇总^[去年给老板准备年会PPT时做了一版最新的，分析结果找不到放哪儿了，找到之后再补]

This repo store some mini-projects based on my inspirations when reading books, papers, or watching TED videos, or talk with friends. Those lingering but fascinating ideas are hard to resist like Siren's song (there is why the repo gets its name). This repo will keep update and improve. In genral, I will share the data and code except some very private data, and comments are very welcome.
